Line Based Programs

ping lets you test whether another machine on the internet is alive, and how long it takes to send a message to that machine and back.
We can ping 8.8.8.8, which is Google’s public DNS server.
control+C tell a program to stop. It will sends the interrupt signal.

sort sorts lines in alphabetical order. It can’t do any sorting until it’s received the last line and that’s what typing control+Dtells it.
control+D stands for end of file and it tells the program that there isn’t any more input coming.
Programs like sort are called line based programs because they read input one line at a time until they reach the end of the input.

Full Screen Interactive Programs less

less displays text one page at a time.

  • scroll down one page: using the D key or by hitting the Space Bar.
  • scroll up one page: using the U key.
  • skip to very last line of the file: using ] (the right angle bracket).
  • get back up to the first line: using [(the left angle bracket).
  • go to a particular line: typing the line number and hitting Enter.

less also has built-in search. You can type slash and then the string you’d like to search for. Press enter or return to execute the search. You can find the next occurrence of the search term by pressing n. Using capital N to go back to the previous occurrence. Search terms are case sensitive. You can use regular expressions.

The Linux File System

The File System Tree

Files and directories are the two most important kinds of objects in a Linux server’s file system.
You can use Quoting and Escaping to add space in a filename.

For Example:

touch 'Great Filename!' # Quoting
touch Great\ Filename\! # Escaping

Linux uses the / forward slash to separate directories, whereas Windows uses the \ backslash.

The Working Directory

pwd prints out the working directory.

Absolute and Relative Paths

The full path is called the absolute path.
A file’s relative path is its location relative to the current working directory.

~ is an abbreviation for your own home directory. You can also ues cd with on arguments to go to your home directory.

Moving and Copying Files

mv is short for move. You can use it to move a file or rename a file.
cp is short for copy. You can use it to copy a file to a directory.
mkdir create new directories.

Globbing

Globbing is for matching files by name in the Unix Shell. It is a kind of pattern matching for file names.

  • * matches any string of characters.
  • ? A single question mark matches any one character.
  • [] List of characters inside square brackets matches any one of the characters inside those brackets.
